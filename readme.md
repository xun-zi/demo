##### 使用setState更新, 不会更新setInterval中state

react每次更新时都会重新运行一次组件，但是setInterval是闭包导致无法更新。

**解决** 

使用useRef

useRef相当于存在于函数之上的外环境，函数每次创建的时候访问的都是一个对象,所以只要不断更新这个对象的函数，同时在setInteval调用这个全局函数就可以解决此问题。
